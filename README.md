# Full-Stack Developer Portfolio

This is a simple, modern portfolio for a full-stack developer. It includes:
- A light/dark mode toggle
- Animations on scroll
- Project modals with detailed descriptions

## How to Use

1. Clone this repository to your local machine.
2. Add your personal details, such as name, skills, and projects.
3. To deploy on GitHub Pages, push the code to a GitHub repository and enable GitHub Pages in the settings.

## Technologies Used
- HTML, CSS, JavaScript
- Responsive Design
- Dark Mode Toggle
- Project Modals
